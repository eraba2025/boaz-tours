# אתר תדמית: לטייל עם בועז

האתר נבנה כדף נחיתה סטטי (HTML/CSS/JS) בעברית ובכיווניות RTL, עם התאמה למובייל ונגישות בסיסית ברמת AA.

## מה כולל המימוש

- Header עם לוגו, ניווט וכפתור "הזמן סיור"
- Hero עם H1/H2 ושני כפתורי פעולה
- אזור אודות עם הטקסט המלא מהאפיון
- גריד סיורים עם כרטיסיות ותגיות מותג
- אזור יצירת קשר עם טופס, טלפון ואימייל לחיצים
- Honeypot נסתר בטופס למניעת ספאם
- כפתור WhatsApp צף בצד שמאל למטה
- Footer עם נוסח זכויות יוצרים

## אבטחה ותשתית בפריסה

1. פרסו את האתר ב-Vercel או Netlify כדי לקבל HTTPS פעיל כברירת מחדל.
2. הפעילו 2FA בחשבונות הניהול:
   - GitHub
   - Vercel/Netlify
3. אין להוסיף שדות תשלום או נתוני אשראי בטופס.
4. אם רוצים קליטת טופס בפועל, ניתן לחבר שירות מאובטח כמו Formspree.
